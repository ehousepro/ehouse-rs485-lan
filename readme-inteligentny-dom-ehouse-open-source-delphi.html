eHouse Inteligentny Dom Darmowe kody źródłowe na licencji GNU. Tylko dla systemu eHouse



Kody źródłowe: oprogramowanie Pokojowych Paneli sterujących w języku Delphi ".Pas" począwszy od wersji 5.0 dla:
  • Paneli graficznych
  • Tabletów
  • Komputerów PC

Oprogramowanie umożliwia sterowanie i Wizualizacje na systemy operacyjne:
  • Windows XP (32 , 64b)
  • Windows Vista (32 , 64b)
  • Windows 7 (32 , 64b)
  • Windows 8 (32 , 64b)

Główne Cechy Oprogramowania:
  • Wizualizacja Automatyczna Online - UDP LAN - Wyświetlanie 1 wybranego sterownika pomieszczeń
  • Sterowanie Online - TCP LAN, WiFi / wysyłanie zdarzeń do sterowników
  • Komunikacja z systemem eHouse i przesyłanie komend - przez WiFi, LAN / TCP
  • Aktualizacja i utrzymanie statusu sterowników we własnej pamięci programu
  • Dekodowanie statusu binarnego ze sterowników
  • Współpraca z wersją systemu eHouse1 pod nadzorem komputera PC, eHouse1 pod nadzorem CommManagera oraz Ethernet Ehouse
